Weather App

The Weather Forecast App is a simple, responsive web application that allows users to get real-time weather information for any location worldwide. The app fetches live weather data from the OpenWeatherMap API, displaying key details such as temperature, humidity, wind speed, and weather conditions (e.g., sunny, rainy, cloudy). Built using HTML, CSS, and JavaScript, the app provides a user-friendly interface for accessing weather forecasts, with the ability to search for weather by city name.

You can check out my Weather app to experience the live functionality and get real-time weather updates!
